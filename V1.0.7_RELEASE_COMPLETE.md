## ðŸ“¦ Quick Start on Your Server

### For Ubuntu Server (18.209.250.77)

**Method 1: Systemd Service (Recommended)**
```bash
ssh ubuntu@18.209.250.77

# Upgrade to v1.0.7
pip install --upgrade glancewatch

# Install as systemd service
curl -sSL https://raw.githubusercontent.com/collinsKemboi/glanceswatch/main/install-service.sh | bash

# Now it runs forever! Close terminal safely.
```

**Method 2: Quick nohup**
```bash
ssh ubuntu@18.209.250.77

# Upgrade to v1.0.7
pip install --upgrade glancewatch

# Start in background
nohup glancewatch > /dev/null 2>&1 &

# Check if running
ps aux | grep glancewatch

# Close terminal - it keeps running!
```
